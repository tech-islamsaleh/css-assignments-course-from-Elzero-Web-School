### Assignment 01

```
<div class="main red">Elzero</div>

<div class="red">Testing</div>
<div class="main red">Elzero</div>

```

- Use the previous structure without modifying it
- The word Elzero is required to be colored red
- Selector / s should be chosen with only one line
